# FAMILY: book.authoring

## SKILLS
### book.structure.plan
- **Purpose**: Map out the structural beats of a book (Act 1-3).
- **Motions**: Define Protagonist -> Map Conflict -> Set Resolution -> Plot Chapter Highs.

### book.character.track
- **Purpose**: Maintain a "Character Bible" to ensure consistency (eyes, height, quirks, backstories).
- **Workflow**: Catalog Character -> Index Traits -> Scan Draft for Discrepancies.

### book.manuscript.compile
- **Purpose**: Bundle all chapter markdown files into a single, formatted manuscript (EPUB/PDF).

---

# WORKFLOW: book.outline_to_draft
1. **Trigger**: "Start Draft Phase for [Book Title]"
2. **Execute**: `book.structure.plan`.
3. **Loop**: Query `book.character.track` -> Write Chapter Section -> Self-Audit for Flow.
4. **Log**: Daily word count and "Atmosphere" score to `MemoryStore`.
